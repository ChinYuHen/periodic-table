# Interactive Periodic Table

A modern, responsive interactive periodic table built with HTML, CSS, and JavaScript.

## Features

- **Interactive Elements**: Click on any element to view detailed information
- **Search Functionality**: Search elements by name, symbol, or atomic number
- **Category Filtering**: Filter elements by chemical category
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, visually appealing design with smooth animations

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- GitHub Pages (Deployment)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start exploring the periodic table!
